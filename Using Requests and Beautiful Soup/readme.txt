Get-links.py

Python code to scrape and store the links of all the restaurants(including all the outlets of a particular restaurant) in Kolkata. Uses requests and beautifulSoup to get this done. Simply run the python code and you will get the output in a file.

get-restaurant-info.py

Python code to get restaurant info from the restaurant's page on Zomato. 
It takes the links of the restaurants as input, scrapes Zomato and provides the following restaurant information as output :  Name,Restaurant ID,Rating,Address,Cuisine,Cost for Two. Uses requests and beautifulSoup to get this done

get-loc.py

Generates a csv file mapping a restaurant to its location in Kolkata
